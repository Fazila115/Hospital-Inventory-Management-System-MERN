# Hospital Inventory Management System
This is a website built with the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This project replaces the traditional spreadsheet-based inventory management with an interactive web application, allowing a single user (Admin) to efficiently manage hospital items and suppliers.

# Tech Stack
React (with Vite)
Redux Toolkit
React Router DOM
Tailwind CSS
Flowbite React
Axios
Toast notifications

# Environment Variables
Create a .env file using the provided .env.example.

# Instalation & Setup Instructions

**Backend**
1. Navigate to the backend folder:  
   cd backend
2. Install dependencies:
   npm install
3. Set up environment variables in .env file
4. Start the server:
    npm run server

**Frontend**
1. Navigate to the frontend folder:
    cd frontend
2. Install dependencies:
    npm install
3. Set up environment variables in .env file
4. Start the React app:
    npm run dev

**Shell Script** 
This project includes a shell script '.sh' to run both frontend and backend servers at once.

Purpose
- Starts the React frontend in development mode.
- Starts the Node.js backend server.
- Saves you from manually running commands in two terminals.

**Run the script from the root directory:**
./.sh
